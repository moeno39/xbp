「プログラミングでこんなことはできないか？」

⇩私たちの班で上がった意見


このような意見が挙がりましたが、最終的に私たちの班では「朝起きられない」や「授業中に眠くなる」という悩みをプログラミングでどうにかできないか考えることにしました。

⇩そしてこのような案を考えました

⇩いくつかのwebサイトを参考にし、作成したソースコード

  1. import schedule
  2. import time
  4. #時間設定
  5. print("スリープBGMを再生する時間を指定してください")
  6. hour = input("時間（hour）:")
  7. minute = input("分（minute）:")
  8. target = f"{hour.zfill(2)}:{minute.zfill(2)}"
  9. print(target+"に再生時間を設定しました")
  11. #defは「定義」ということ
  12. def job():
  13.     import webbrowser
  14.     #再生したい動画を指定
  15.     webbrowser.open("https://youtu.be/8GYpz-QesEQ")
  17. schedule.every().day.at(target).do(job)
  19. #時間待ち
  20. while True:
  21.   schedule.run_pending()
  22.   time.sleep(60)

このプログラムでは自分で設定した時間に眠くなるYouTubeを流すことができます。YouTubeが流れることによって強制的に眠くなり、生活習慣の改善と授業中の睡魔対策にもつながります。
