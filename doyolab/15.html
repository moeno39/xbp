Shopの項目を作成しました。

オンラインショップのページに飛べるボタンの作成
display: inline-block; を使用して、ボタンの高さと幅を調節。 

                HTML
                    <div class="shop__link">
                     <a class="shop__link-btn" href="" target="_blank" rel="noopener noreferrer">Online Shop</a>
                     <p class="shop__link-caption">
                        丁寧に焙煎したコーヒー豆やオリジナルグッズを、オンラインでご購入頂けます。<br>
                     </p>
                    </div>
                
            
                CSS
                    .shop__link {
                        margin-top: 11vh;
                        text-align: center;
                      }
                      
                      .shop__link-btn {
                        width: 40vw;
                        display: inline-block;
                        padding: 0.8vw 0 0.8vw 0;
                        font-size: 1.5rem;
                        background-color: #B98A67;
                        text-decoration: none;
                      }
                      
                      .shop__link-btn:link, .shop__link-btn:visited {
                        color: #EDECE7;
                      }
                      
                      .shop__link-caption {
                        font-size: 1.0rem;
                        line-height: 1.8;
                        margin-top: 3.5vh;
                      }
