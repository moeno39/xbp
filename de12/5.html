⇩私たちの班であがった意見


このような意見があがりましたが、最終的に私たちの班では「朝起きられない」や「授業中に眠くなる」という悩みをプログラミングでどうにか解決できないか考えることにしました。

⇩いくつかのwebサイトを参考にし、作成したソースコード

  1. import schedule
  2. import time
  4. #時間設定
  5. print("スリープBGMを再生する時間を指定してください")
  6. hour = input("時間（hour）:")
  7. minute = input("分（minute）:")
  8. target = f"{hour.zfill(2)}:{minute.zfill(2)}"
  9. print(target+"に再生時間を設定しました")
  11. #再生したい動画を指定
  12. print("再生したいスリープBGMのURLを貼ってください")
  13. movie = input("URL :")
  14. print(target+"に指定したBGMを再生します")
  16. #defは「定義」ということ
  17. def job():
  18.     import webbrowser
  19.     webbrowser.open(movie)
  21. schedule.every().day.at(target).do(job)
  23. #時間待ち
  24. while True:
  25.   schedule.run_pending()
  26.   time.sleep(60)

このプログラムでは自分で設定した時間に指定した動画を流すことができます。聴くと眠くなる動画を設定することによって強制的に眠気が襲ってくるので、生活習慣の改善と授業中の睡魔対策にもつながります。



⇩実行してみるとこんな感じ

実際に時間になると動画が勝手に再生されます。

問題点は、常にパソコンを開いていないと動画が再生されないことです。
